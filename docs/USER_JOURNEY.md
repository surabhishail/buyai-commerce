# User Journey

## Scenario: Sarah Wants Protein Powder

### Stage 1: Search
**User:** "I need protein powder under $30"

**AI Response:** Shows 3 products with prices, images, ratings

**Time:** 3-5 seconds

---

### Stage 2: Selection
**User:** Clicks first product

**AI Response:** Order summary with total, shipping, tax

**Time:** 1-2 seconds

---

### Stage 3: Payment
**User:** Enters card details in Stripe form

**System:** Processes payment, creates order

**Time:** 2-4 seconds

---

### Stage 4: Confirmation
**AI Response:**
```
✅ Order placed successfully!
Order #12345
Estimated delivery: Jan 25-27
```

**Total Journey Time:** 2-3 minutes
